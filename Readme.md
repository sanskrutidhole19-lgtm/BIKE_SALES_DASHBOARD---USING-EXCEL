<h1 align="center">📊 Interactive Bike Sales Analytics Dashboard</h1>

A complete Excel-based analytics project focused on visualizing customer purchase behavior through an interactive and user-friendly dashboard.

The project demonstrates practical data analytics skills including data cleaning, KPI tracking, pivot analysis, and dashboard reporting using Microsoft Excel.

---

🎯 Project Goals

This dashboard was created to:

* Analyze bike purchasing trends
* Discover customer demographics influencing sales
* Build interactive reports for better visualization
* Practice real-world Excel dashboard development

---

🧾 Dataset Overview

The dataset includes customer records with various attributes:

| Column Name      | Description               |
| ---------------- | ------------------------- |
| Age              | Customer age              |
| Gender           | Male/Female customer data |
| Income           | Annual income details     |
| Education        | Educational background    |
| Occupation       | Customer profession       |
| Region           | Customer location         |
| Commute Distance | Distance traveled daily   |
| Purchased Bike   | Bike purchase status      |

📌 Key Dashboard Features

🔹 Dynamic Filters

Slicers allow users to interactively filter dashboard data based on categories.

🔹 Income vs Purchase Analysis

Compares average customer income with bike purchasing behavior.

🔹 Age-Based Customer Insights

Highlights which age categories contribute most to bike sales.

🔹 Commute Distance Evaluation

Analyzes how daily travel distance affects purchase decisions.

🔹 Region-Wise Sales Understanding

Provides insights into customer buying behavior across regions.

📉 Insights Generated

* Middle-aged customers showed higher purchase rates.
* Customers with shorter commute distances were more likely to purchase bikes.
* Income level influenced buying decisions.
* Regional variations affected overall sales trends.
